Sobre meu aprendizado na Trybe

Meu aprendizado na Trybe se iniciou com o conteúdo de Unix e Bash, desse modo eu fui capaz de usar comandos no terminal para navegar e modificar diretórios e arquivos além de monitorar processos, aprendi também como utilizar o Git e o GitHub para versionar arquivos em um repositório, utilizando branchs, commits e pull requests. Aprendi como estruturar o conteúdo de uma página Web com HTML semântico e estilizá-la usando CSS.
